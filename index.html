My CAD Projects

Game Controller

December 2021

I designed a game console controller based off the design of the famous Play Station 4 Controller. This project taught me about surfacing and loft techniques which are used to make complex curvature in a simple way.

Pocket Lamp

February 2018

Created an aesthetic pocket lamp/flash light for an injection molding environment. I implemented proper draft angling, lip and grooves, as well as ribs to allow for smooth manufacturability.

Detergant Bottle

September 2017 - January 2018

The idea of a detergant bottle is extremely simple, yet difficult to design using SolidWorks. Using surfacing commands such as boundary surface and filled surface paired with 3D sketches, I was able to create a bottle with close to perfect curvature.
